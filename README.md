# Learn_navigation_w_TurtleBot3
Разработка учебного курса для изучения задач навигации с помощью робота TurtleBot3 на основе ROS

Этап 1. Введение в ROS и архитектуру системы

Содержание:
 1. Установка и настройка среды.
 2. Изучение структуры workspace.
 3. Работа с узлами, топиками, сообщениями.
 4. Запуск модели TurtleBot3 в симуляции.


Этап 2. Работа с сенсорами и одометрией

Содержание:
 1. Подписка на данные LiDAR.
 2. Работа с одометрией.
 3. Визуализация данных в RViz.
 4. Анализ погрешностей движения.


Этап 3. Построение карты 

Содержание:
 1. Запуск SLAM-пакета.
 2. Построение карты помещения.
 3. Сохранение карты.
 4. Анализ качества карты.


Этап 4. Локализация на карте

Содержание:
 1. Загрузка ранее созданной карты.
 2. Настройка AMCL.
 3. Анализ сходимости локализации.
 4. Работа с неопределённостью положения.


Этап 5. Планирование траектории и навигация

Содержание:
 1. Подключение планировщика пути.
 2. Настройка глобального и локального планирования.
 3. Отправка целевой точки.
 4. Анализ движения и обход препятствий.


Этап 6. Интеграционный проект

Содержание:
 1. Построение карты.
 2. Локализация.
 3. Автономное перемещение по нескольким целям.
 4. Анализ стабильности работы системы.
